# TIC TAC TOE GAME

## Features
- Classic Tic Tac Toe design.
- Play against the computer or a second player.
- Clear and intuitive user interface.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vinayan00123/TIC-TAC-TOE-GAME.git
   ```
2. Navigate into the project directory:
   ```bash
   cd TIC-TAC-TOE-GAME
   ```
3. Install the required dependencies (if any).

## Gameplay Modes
- **Single Player:** Play against the computer.
- **Multiplayer:** Play against a friend on the same device.

## Upgrade Details
- Future versions will include:
  - Enhanced AI for single-player mode.
  - Online multiplayer feature.
  - Customizable board sizes.
